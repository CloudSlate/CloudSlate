# Guide: Creating Realistic Blog Posts in Sanity CMS

Now that dummy posts are removed, here's how to create high-quality, realistic blog posts in Sanity CMS.

## üéØ Access Your Sanity Studio

1. Go to: **https://cloudslate.sanity.studio**
2. Login with your Sanity account
3. You'll see the "CloudSlate CMS" interface

## üìù Creating Your First Realistic Post

### Step 1: Click "Create new" ‚Üí "Blog Post"

### Step 2: Fill in Required Fields

#### **Title** (Required)
- Make it clear and descriptive
- Include keywords naturally
- Keep it under 70 characters for SEO
- **Example**: "How to Build a Modern Blog with Sanity CMS and Cloudflare Pages"

#### **Slug** (Required)
- Auto-generated from title (or customize)
- Should be URL-friendly (lowercase, hyphens)
- **Example**: `how-to-build-modern-blog-sanity-cloudflare`

#### **Excerpt** (Required - 25-160 characters)
- Compelling preview text
- Should summarize the post
- Include a call-to-action if possible
- **Example**: "Learn how to build a fast, SEO-optimized blog using Sanity CMS for content management and Cloudflare Pages for hosting. Perfect for developers and students."

#### **Content** (Required)
Use the rich text editor to create well-structured content:

**Best Practices:**
- Start with an engaging introduction
- Use headings (H2, H3) to structure content
- Break up text with paragraphs
- Add images where relevant
- Include code examples if technical
- End with a conclusion or call-to-action

**Example Structure:**
```
Introduction (Hook the reader)

## Main Section 1
Content here...

## Main Section 2
Content here...

### Subsection
More details...

## Conclusion
Wrap up with key takeaways
```

#### **Author** (Required)
- Default: "Muhammad Khuhro"
- Can be customized per post

#### **Published Date** (Required)
- Auto-set to current date
- Can be changed to schedule posts

#### **Category** (Required)
Select from:
- Cloud Computing
- Web Development
- Tutorials
- Technology
- DevOps
- Programming

#### **Tags** (Optional but Recommended)
- Add 3-5 relevant tags
- Helps with search and organization
- **Examples**: "Sanity CMS", "Cloudflare", "Blogging", "Tutorial"

#### **Featured** (Optional)
- Toggle ON for important posts
- Featured posts appear in the homepage featured section

#### **Featured Image** (Recommended)
- Upload high-quality image
- Or use external URL (e.g., Unsplash)
- Add descriptive alt text for accessibility
- **Recommended size**: 1200x630px for social sharing

#### **Read Time** (Required)
- Estimated reading time in minutes
- Calculate: ~200 words per minute
- **Example**: 1000 words = 5 minutes

#### **SEO Settings** (Optional but Recommended)

**Meta Description:**
- Custom meta description (25-160 characters)
- If empty, excerpt will be used
- Should be compelling and include keywords

**Keywords:**
- Add relevant keywords for SEO
- 5-10 keywords recommended
- **Example**: "Sanity CMS", "Cloudflare Pages", "Blog Setup", "Content Management"

### Step 3: Publish

Click **"Publish"** in the top right corner. Your post will immediately appear on your website!

### Unpublishing Posts
- Edit the post
- Click "Unpublish" instead of "Publish"
- Post will be removed from website but saved in Sanity
